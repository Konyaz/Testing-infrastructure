# Установка Docker




Проверяем, установлен ли Docker командой:
```bash
docker --version
```

Устанавливаем Docker Скриптом с помощью curl командой:
```bash
sudo curl -fsSL https://get.docker.com -o get-docker.sh
```
Проверяем установку Docker командой:
```bash
docker --version
```
